我修改的一个 Wordpres 的视频插件, 主要因为之前随便搜了一下没有找到支持国内视频共享网站的视频播放插件, 所以照着 Wordpress Youtube Plugin 改了一个, 其实挺简单的, 依样画葫芦就是了. 把改好的放上来算是服务大众吧...

插件说明
名称: aVideo
说明: Allen's Wordpress Video Plugin, 支持主流中文视频分享网站的 Flash 视频播放插件.
版本: 0.9
作者: Allen Hsu
网址: http://www.imallen.com/
插件: http://www.imallen.com/blog/works/2007/09/avideo-allens-wordpress-video-plugin/

安装方法
将 avideo.php 上传至 wordpress 根目录下的 wp-content/plugins/ 目录, 然后在后台 Plugins 激活 aVideo 即可使用.

使用方法:

YouTube:
[youtube id]
其中 id 为播放页面地址 http://www.youtube.com/watch?v=<strong>fY4Epc2XSGc</strong> 中加粗部分

土豆网:
[tudou id]
其中 id 为播放页面地址 http://www.tudou.com/programs/view/<strong>Y2dE5FH-LwE</strong>/ 中加粗部分

QQ 视频:
[qqvideo id]
其中 id 为播放页面地址 http://video.qq.com/videopl.htm?v=<strong>2HTaRoQGRNr</strong>&amp;i=1 中加粗部分

Mofile:
[mofile id]
其中 id 为播放页面中论坛代码 http://tv.mofile.com/cn/xplayer.swf?v=<strong>A41L6TYX</strong>&amp;type=list&amp;autoplay=0&amp;nowSkin=0_0 中加粗部分

UUme:
[uume id]
其中 id 为播放页面地址 http://www.uume.com/play_<strong>zdLLLAqLedut</strong> 中加粗部分

优酷:
[youku id]
其中 id 为播放页面地址 http://v.youku.com/v_show/id_ce00<strong>XODI1ODQ3Ng==</strong>.html 中加粗部分
注意: 带 = 号但是不带 00 以及 00 以前的字符

其他 Flash:
[flash id]
其中 id 为 Flash 地址

专用播放器已都根据默认大小设置, 通用 Flash 播放器大小默认为 400 * 300.

有任何疑问或建议的话欢迎到 http://www.imallen.com/blog/works/2007/09/avideo-allens-wordpress-video-plugin/ 留言.
